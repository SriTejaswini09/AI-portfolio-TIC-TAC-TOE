About This Project

Hello! I'm Sri Tejaswini, an SRM student exploring the field of Artificial Intelligence. This portfolio project features a simple yet effective Tic Tac Toe game that allows you to play against a computer using AI logic. The goal of this assignment was to apply AI concepts in a fun and interactive way.

The AI in this Tic Tac Toe game uses a simplified version of the Minimax algorithm. It scans the board to determine winning opportunities, blocks the player from winning, and chooses optimal moves based on predefined rules. This introduces basic AI logic used in game strategies.

The decision-making is implemented using JavaScript and demonstrates how a machine can play against a human using logical evaluation.
